# Pokemon Studio 中文字体

**构建方法**：将 Pokemon Studio 的默认字体与中文等线字体进行合并，以支持中文字符显示。

## 如何使用
1. **下载字体**：直接下载**ChineseFonts**中的字体文件。
2. **应用到 Pokemon Studio**：
   - 在游戏文件目录**Fonts**替换默认字体。
   > **注意**：部分特殊字符可能显示异常。
